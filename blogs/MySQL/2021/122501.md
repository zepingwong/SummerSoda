## ***背景***

- **操作系统**：Ubuntu 20.04
- **软件**：MySQL 8.0

### ***问题描述***

在MySQL 8.0 跳过权限验证（`skip-grant-tables`）修改 root 用户密码 /配置远程访问时，出现错误：

```
ERROR 1290 (HY000): The MySQL server is running with the --skip-grant-tables option so it cannot execute this statement
```

## ***解决办法***

执行以下SQL语句，刷新权限：
```
flush privileges;
```

再次修改密码或进行授权，轻松解决。
