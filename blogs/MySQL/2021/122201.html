Debian GNU/Linux apt 命令行安装 MySQL 8.0

WANG Zeping 2021/12/22 MySQL

Debian 系列的 Linux 系统理论上都可以使用，本文桌面操作系统以 deepin 20 系统为例进行演示。

# mysql-apt-config

下载安装 mysql-apt-config.deb (opens new window)

# 下载

# 官网下载

不用登录，直接下载即可

# wget下载

没有桌面操作系统，或者习惯命令行的同学可以用 wget 进行下载，软件版本请以官网为准。

wget https://dev.mysql.com/get/mysql-apt-config_0.8.22-1_all.deb
1

# 安装

# 桌面安装

安装刚刚下载的 deb 软件包

由于这个库是为 Debian 和 Ubuntu 准备的，在检测到系统不符时，需要选择系统 Debian和Ubuntu版本比较

Ubuntu版本号 Ubuntu代号 debian版本号 debian代号
14.04 LTS Trusty 8 jessie
16.04 LTS Xenial 9 strech
18.04 LTS Bionic 10 buster
20.04 LTS Focal 11 Bullseye

查看 deepin 版本与 Debian 版本 命令1

lsb_release -a
1

输出

No LSB modules are available.
Distributor ID:	Deepin
Description:	Deepin 20
Release:	20
Codename:	n/a
1
2
3
4
5

命令2

cat /etc/issue
1

输出

Deepin GNU/Linux 20 \n \l
1

命令3

cat /etc/debian_version
1

输出

10.3
1

确定 debian 大版本为 10 ，因此输入 1 ，选择 debian buster ，然后输入 4 选择 OK 默认安装。完成后mysql-apt-config 的 GUI 配置窗口会自动关闭，此时 mysql-apt-config 的配置就完成了。

# 命令行安装

sudo dpkg -i mysql-apt-config_0.8.22-1_all.deb
1

选择 OK 然后确认即可

# 更新包

mysql-apt-config 安装成功后，更新包

sudo apt-get update
1

# 安装

通过 apt-get 安装 MySQL

sudo apt-get install mysql-server
1
