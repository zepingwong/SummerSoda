## ***问题描述***

Windows、Linux系统下安装Anaconda 后，打开cmd或终端会自动加载base环境，怎么破？

## ***正常解决方案（推荐）***

其实这个问题在完成安装时，已经给出了解决办法：

![Snipaste_2021-12-06_15-27-53.png](https://s2.loli.net/2022/05/08/zKdHR7GPLV25svT.png)

命令行运`conda`配置指令即可：
```shell
conda config --set auto_activate_base false
```

## ***次级解决方案（Linux）***

修改 `～/.bashrc`

```shell
# vim编辑～/.bashrc
vim ~/.bashrc
```

在最后加入：

```shell
conda deactivate
```

>这句指令就是退出当前环境

保存退出后，执行

```shell
source ~/.bashrc
```
