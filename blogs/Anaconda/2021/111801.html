Ubuntu 命令行安装 Anaconda3（root用户）

WANG Zeping 2021/11/18 PythonUbuntuAnaconda

# 背景

特定情景下需要root用户安装Anaconda，同时，普通用户需要分享Anaconda环境。这里整理一下root用户和普通用户共享anaconda环境的笔记。

# 一、下载Anaconda

# 方法 1：官网

Anaconda官网 (opens new window)下载（比较费时）

# 复制下载地址

# wget下载

wget https://repo.anaconda.com/archive/Anaconda3-2021.11-Linux-x86_64.sh
1

# 方法 2：清华源

在清华大学开源软件镜像站 (opens new window)下载

# 复制下载地址

选择需要的Anaconda版本，右键复制链接地址。

# wget下载

wget https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/Anaconda3-2021.11-Linux-x86_64.sh
1

# 下载

# 二、安装 Anaconda

# 1. 进入目录

进入刚刚下载的文件所在的目录。

# 2. 运行 .sh 文件：

使用sudo bash root权限运行 .sh 文件。

sudo bash Anaconda3-2021.11-Linux-x86_64.sh
1

# 3. 阅读并接受许可

敲回车阅读许可信息，输入 yes 接受许可条款。

# 3. 修改安装目录

修改软件即将安装的位置，修改为 /opt/anaconda3 ，回车开始安装。

# 4. 初始化conda

文件解压完成后，收到是否初始化conda的提示信息，输入 yes，敲回车继续。

# 5. 完成

# 三、普通用户配置环境

普通用户登陆后，需要执行以下指令使conda的路径等系统环境变量信息写入当前用户的 ~/bashrc 文件中：

/opt/anaconda3/bin/conda init bash
1

刷新环境变量

source .bashrc
1
