# 常用变量

# title

文章标题，放弃通过一级目录定义标题的方式，改在 Front Matter 中定义。

当 title 中出现英文冒号（:）时，需要使用引号设置，比如 title: 'Error: Solve the problem'

# date

文章创建日期，格式 2019-08-08 或 2019-08-08 08:08:08。

设置日期 时间格式时，需要使用引号

# sidebar

是否开启页面内侧边栏。

# categories

所属分类。

# tags

所属标签。

# keys

文章加密密码。

# publish

文章是否发布。

# sticky

  • description: 文章置顶。
  • type: number
  • sort type: 降序，可以按照 1, 2, 3, ... 来降低置顶文章的排列优先级
